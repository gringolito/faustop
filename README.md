# faustop

## Player com frases aleatórias do Fausto Silva

#### Install
Para instalar no sistema Linux

    mkdir build
    cd build
    cmake ..
    cmake --build . --target install
    
#### Package
Para criar pacote deb

    mkdir build
    cd build
    cmake ..
    cpack -G DEB .

#### Usage
Executar uma frase aleatória
faustop
